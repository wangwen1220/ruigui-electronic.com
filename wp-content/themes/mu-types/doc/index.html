About Mu TypesTOP

Mu Types is a very powerful theme which suits both, users with no programming background as well as advanced developers. It will help you build your site in no time to your liking with minimal effort. It empowers you to set color to elements such as links, text, menu links, etc. The theme also features 3(Three) Default Home page and single page layout for you to choose from.

This theme is designed for use with WordPress 3.2 or higher. The code was written to be backwards compatible where possible, however it is recommended you use version 3.2 or higher. If you are not using WordPress 3.2 or higher and you can upgrade your site I recommend you do this before installation.

Although every effort has been made to ensure the theme is bug free and easy to use there is always the chance for errors. If you do find a problem please notify us immediately through the ThemeForest account page so we can ensure it gets fixed as quickly as possible.

The main features:

  • Unlimited Colors with colorpicker.
  • Custom Background: upload the image by yourself.
  • Powerful and Easy theme options.
  • 6 custom post types ready: portfolio, product, download, event, team, slidershow.
  • 2 layouts and 2 sortable(classes and quick sand.) ready for portfolio.
  • 2 layouts ready for blog.
  • 8 custom widgets ready: ads, tweets, flickr, blog, download, event, comment, search.
  • SEO has been made a priority when designing the theme.
  • SEO for single page. (Title tags, Keywords, Description Meta, Robots.)
  • A lot of shortcodes collection are ready for you.
  • Many page templates layout.
  • Automated thumbnail resizing.
  • Full Localisation Support: de_DE, en_US, es_ES, it_IT, nl_NL.(contains .po/.mo files)
  • Support Wordpress MU
  • HTML5 And CSS3 .
  • Browser compatibility(IE8 ,IE9, Firefox 3-8, Chrome, Safari, Opera. )

InstallationTOP

Installing the theme can be done two ways. You can upload the theme ZIP file using the built in WordPress theme upload, or copy the files through an FTP client.

1. Using WordPress Theme Upload

Be sure to extract the file "mu-types-clean-business-wordpress-theme.zip" from the ThemeForest download before uploading. Using the ThemeForest ZIP file directly will result in a "Missing Style Sheet" error. And don't change the folder name.(mu-types), So you should upload the file (mu-types.zip)

  1. Login to your WordPress admin.
  2. In the "Appearance => Themes" menu click the tab "Install Themes"
  3. At the top of the page click, "Upload", then click the file input to select a file.
  4. Select the zipped theme file, "mu-types.zip", and click "Install Now"
  5. After installation you will receive a success message confirming your new install.
  6. Click the link "Activate"

2. Uploading by FTP

  1. Login to your FTP server and navigate to your WordPress themes directory. Normally this would be "wp-content/themes"
  2. Extract the files "cross-apple.zip" from the zipped theme.
  3. Copy the folder "mu-types" to your themes directory.
  4. After the files finish uploading, login to your WordPress admin.
  5. In the "Appearance" menu click "Themes"
  6. Click "Activate" for the theme "Mu Types"
